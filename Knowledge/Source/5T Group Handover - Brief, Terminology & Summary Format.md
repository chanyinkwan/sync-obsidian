## 3. 晨會紀要格式(供日後「錄音→我整理」用)

英文版(Ziyi 的和記常用):
```
6.24 Morning Meeting Minute (Hutchison part)
1. <topic / action> _<owner>
2. ...
```
中文版(1組常用):
```
24/06 1组晨会内容及待办（和记）：
1. <事項> -<负责人>
2. ...
```
> 慣例:每條 = 一個議題 + 負責人(用 `_owner` 或 `-负责人`);標題含日期 + 組別 + 帳戶。**日後你給我逐字稿,我就照這格式輸出 + 把 ⚠ 術語標出來讓你核對。**
